# Kraftstoffverbrauchsanalyse

Python-Projekt zur Analyse von Kraftstoffverbrauch im Logistikumfeld.

## Funktionen
- Einlesen von CSV-Daten
- Berechnung des Verbrauchs (l/100km)
- Durchschnitt berechnen

## Technologien
- Python
- CSV

## Example Output
Verbrauch pro Fahrt:
8.33  
9.00  
8.00  
8.33  

Durchschnitt: 8.42 l/100km
